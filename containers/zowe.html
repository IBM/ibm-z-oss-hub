IBM Z Distribution for Zowe

Containerized Zowe™ provides Zowe services in popular container formats providing an easier way for customers to leverage Zowe to modernize their current mainframe environment.

Besides the strength that Zowe brings you in the modernized way, Zowe container:

  • allows you to run Zowe in other system and have orchestration provided by Kubernetes
  • is as secure as using Zowe on z/OS
  • keeps strength of high availability and high scalability in a fully automated way
  • helps simplify Zowe installation & configuration

See zowe.org for more information
