onnx-mlir

ONNX-MLIR is an open-source project for compiling ONNX models into native code. It is built on top of Multi-Level Intermediate Representation (MLIR) compiler infrastructure. A compiled model with this image is built to run either on the IBM Z CPU processor, or on the IBM Z Integrated Accelerator for AI which is available on IBM z16 or greater machines.

Complete documentation is at ONNX-MLIR Homepage and ONNX-MLIR GitHub

IBM zSystems specific documentation and usage examples is available at IBM Z Deep Learning Compiler Documentation and Examples.

See ONNX-MLIR Homepage for more information

This image is built by IBM to run on the IBM Z architecture and is not affiliated with any other community that provides a version of this image.
