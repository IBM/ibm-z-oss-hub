Usage Notes

Exposed Ports: 3306

Start your container binding the external port 3306. Replace {assign-pass-here} with the desired password you would like your root user to use.. 

docker run -d --name mariadb -p {mariadb-port}:3306 -e MARIADB_ROOT_PASSWORD={assign-pass-here} icr.io/ibmz/mariadb:{tag}

Further documentation can be found at mariadb.com
