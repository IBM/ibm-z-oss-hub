Containerized Workloads

The images in this registry have several key characteristics:

  • The IBM team has built these images from source - they are not assembled from images hosted at other registries.
  • For images that include a build test, the IBM team runs these tests, and ensures that the images start as documented. No other tests have been performed.
  • These images have been scanned for known vulnerabilities and a report can be seen by clicking on the issues listed under the Security status on the details page for each image.
  • The sha256 hash for each image has been published for reference purposes. These hash strings can be comapred after pull to verify image contents. Although not required, we recommend using these strings when pulling any of these images.
